# AI Native Eval Project Directory

This directory is source-controlled project configuration space for `ai-native-eval`.

Commit shared configuration here, such as:

```text
.ai-native-eval/config.json
```

Generated run outputs belong under:

```text
.ai-native-eval/artifacts/
```

`artifacts/` is ignored by git. Promote any reviewed report that should become durable evidence into a stable committed folder such as `self-evaluations/` or project docs.
